# TAURA SEO — Skill de Otimizacao para Buscadores

## Resumo
Otimiza artigos e paginas TAURA para mecanismos de busca. Analisa keywords, concorrencia e volume de pesquisa no nicho de peptideos e saude.

## Responsabilidades
1. **Keyword Research** — Identificar keywords de alto volume e baixa concorrencia
2. **Meta Descriptions** — Escrever descriptions otimizadas (max 160 chars)
3. **Structured Data** — Sugerir schema.org markup (Article, FAQ, HowTo)
4. **Internal Linking** — Conectar artigos relacionados
5. **Title Tags** — Otimizar titulos (max 60 chars, keyword no inicio)
6. **Content Gaps** — Identificar topicos nao cobertos pela concorrencia

## Checklist SEO por Artigo
- [ ] Title tag com keyword principal (max 60 chars)
- [ ] Meta description unica (max 160 chars)
- [ ] H1 unico, H2/H3 hierarquicos
- [ ] Keyword principal nos primeiros 100 palavras
- [ ] Imagens com alt text descritivo
- [ ] Links internos para artigos relacionados
- [ ] Pelo menos 1 link externo para fonte confiavel (PubMed, etc.)
- [ ] URL amigavel (slug curto e descritivo)
- [ ] Schema markup (Article + FAQ se aplicavel)

## API
Usa os mesmos endpoints de `/api/articles` da skill taura-content.
Foco: analisar e melhorar campos `title`, `slug`, `meta_description`, `tags`, `body_html`.
